
# KoreanTimeBlocker
Alarm system considering preparation time and time to destination

## Getting Started
You must install pod file

#### GoogleMaps
<code>
  pod 'GoogleMaps'
</code>
<br>

#### GooglePlaces
<code>
  pod 'GooglePlaces'
</code>
<br>
